## Controles del Juego

<CardGrid stagger>
  <Card title="Movimiento" icon="right-arrow">
    Usa WASD o las Flechas para navegar por el campo de batalla y esquivar ataques enemigos.
  </Card>
  <Card title="Combate" icon="rocket">
    Clic Izquierdo o Espacio para atacar. Sincroniza tus golpes cuidadosamente para maximizar el daño.
  </Card>
  <Card title="Habilidades Especiales" icon="star">
    Presiona E o Clic Derecho para activar habilidades especiales cuando estén disponibles.
  </Card>
  <Card title="Menú de Pausa" icon="setting">
    Presiona ESC para pausar el juego y acceder a configuraciones o volver al menú principal.
  </Card>
</CardGrid>

## Consejos del Juego

<CardGrid stagger>
  <Card title="Mantente en Movimiento" icon="rocket">
    El movimiento constante es clave para sobrevivir. Quedarse quieto te convierte en un blanco fácil para los enjambres.
  </Card>
  <Card title="Prioriza Amenazas" icon="warning">
    Concéntrate primero en los enemigos más peligrosos. Algunos insectos infligen más daño o se mueven más rápido que otros.
  </Card>
  <Card title="Administra Recursos" icon="star">
    Usa las habilidades especiales sabiamente. Guárdalas para situaciones abrumadoras o encuentros difíciles con jefes.
  </Card>
  <Card title="Aprende Patrones" icon="open-book">
    Cada oleada tiene patrones predecibles. Estudia el comportamiento enemigo para anticipar ataques.
