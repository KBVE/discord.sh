## ゲームコントロール

<CardGrid stagger>
  <Card title="移動" icon="right-arrow">
    WASDまたは矢印キーを使用して戦場を移動し、敵の攻撃を回避します。
  </Card>
  <Card title="戦闘" icon="rocket">
    左クリックまたはスペースで攻撃。タイミングを計って攻撃し、ダメージを最大化しましょう。
  </Card>
  <Card title="特殊能力" icon="star">
    EキーまたはRight Clickで利用可能な特殊能力を発動します。
  </Card>
  <Card title="ポーズメニュー" icon="setting">
    ESCキーを押してゲームを一時停止し、設定にアクセスするか、メインメニューに戻ります。
  </Card>
</CardGrid>
